# Digital Wedding Card 💍

A mobile-first digital wedding invitation website created for the wedding celebration

---

## About the Project

**Digital Wedding Card** is a responsive digital wedding invitation built using HTML, CSS, and JavaScript.

The website follows a **Romantic Garden** theme with soft floral backgrounds, champagne-gold details, burgundy accents, elegant typography, and smooth animations.

It is designed mainly for mobile users while remaining responsive on desktop devices.

---

## Features

- Animated envelope opening
- Full wedding poster preview
- Automatic poster countdown
- Button to view the wedding poster again
- Arabic Bismillah introduction
- Couple and parent information
- Calendar-style wedding date section
- Live wedding countdown timer
- Wedding event timeline
- Venue photo
- Google Maps navigation
- Waze navigation
- Copy venue address button
- WhatsApp contact buttons
- Direct phone call buttons
- Floating flower petals and leaves
- Smooth scroll animations using AOS
- Background wedding music
- Floating music play and pause control
- Final thank-you section
- Responsive mobile-first design
- Reduced-motion accessibility support

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- Google Fonts
- GitHub Pages

No frameworks, backend, or database are required.
